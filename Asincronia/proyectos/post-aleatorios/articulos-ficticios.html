Solicitudes HTTP

Aqui puedes obtener y enviar datos a través de solicitudes HTTP.

Al presionar obtener, se obtendrán los artículos ficticios, que tienen un titulo y un parrafo. Estos seran renderizados en el DOM, como targetas.

Al presionar enviar, se enviarán los datos a través de una solicitud POST. Estos los puedes ver en la consola.

function enviarPeticion( metodo, url, data) {
  return fetch(url, {
    method: metodo,
    body: JSON.stringify(data),
    headers: {"Content-type": "application/json"},
  }).then(response => {
    return response.json()
  })
}

async function traerPost() {
