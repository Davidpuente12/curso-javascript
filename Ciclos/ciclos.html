Ciclos

En JavaScript, los ciclos (o bucles) son estructuras que permiten ejecutar un bloque de código repetidamente, mientras se cumpla una condición. Son esenciales para automatizar tareas repetitivas, recorrer listas, validar datos, y mucho más.

Ciclo for

(para) es un metodo que crea un bucle que consiste en tres expresiones opcionales, encerradas en parentesis y separadas por puntos y comas, seguidas de una sentencia ejecutada en un bucle. permite recorrer elementos dentro de una lista o array.

Sintaxis

for ( expresion inicial ; condicion ; incremento ){
   sentencia
}

  • Exprecion inicial define una variable de control (comúnmente 'i'), asignando un valor inicial. Por ejemplo, let i = 0.
  • Condicion evalúa la condición que debe cumplirse para que el loop continúe. Normalmente, se compara la variable de control con una longitud. Un ejemplo es i < lista.length.
  • Incremento se ejecuta al final de cada iteracion del bucle, incrementando el valor de la variable en cada iteracion en uno (i++).
  • Sentencia este es el bloque de codigo que se ejecutara todas las veces que la condicion sea verdadera.
let numero = 10

for ( let i = 0 ; i <= 10 ; i ++ ){
  console.log(i)
  console.log(numero ++)
}

Recorrer Arrays
let frutas = ["mandarina","cambur","naranja","sandia"]

for ( let i = 0 ; i < frutas.length ; i ++ ){
  console.log(frutas[i])
}

Ciclo for of

El método loop for-of es una herramienta que te permite recorrer los valores de objetos iterables como (arrays, string, Map, Set) que implemetan Symbol.iterator, Gracias a su estructura, el for-of nos permite ejecutar código de manera repetitiva para cada elemento presente en estas colecciones.

Sintaxis

for ( variable of objeto iterable ){
   sentencia
}

  • variable esta almacena temporalmente a cada elemento del objeto iterable durante cada iteracion del bucle.
  • objeto iterable representa la lista de objetos iterables

let canasta = ["harina","leche","pan","queso"]

for ( let item of canasta ){
  console.log(item)
}

iterar string
let string = palabras

for ( let item of string ){
  console.log(item + " " )
}


Comienza con la palabra reservado for, añade parantesis (), donde se define una variable que representa cada elemento, el termino of seguido del objeto iterable (array) a recorrer. Finalmente entre llaves {} ejecuta el bloque de codigo para cada elemento de la lista.

Ciclo for in

Es una estructura de bucle que recorre las propiedades enumerables de un objeto, en lugar de iterar sobre valores como un for...of (elementos iterables como arreglos y cadenas) , aquí iteras sobre claves (nombres de propiedades), los objetos en JavaScript poseen propiedades, cada una con un valor asociado, a las cuales podemos acceder.

Sintaxis

for ( variable in objeto ){
   codigo a ejecutar
}

  • variable es la que representara cada propiedad del objeto durante la iteracion
  • objeto es el objeto que estamos iterando

const canasta = {
  harina: 5,
  leche: 3,
  pan: 2,
  queso: 1
}

for ( let producto in canasta ){
  console.log(producto) // Imprime la propiedad
  console.log(canasta[producto] ) // Imprime el valor de la propiedad
  console.log(producto + canasta[producto] )
}

Ciclo while

(mientras) crea un bucle que ejecuta una sentencia especificada, mientras cierta condicion se evalue como verdadera, dicha condicion es evaluada antes de ejecutar la sentencia. Esta técnica permite realizar iteraciones eficientes, ejecutando una secuencia de instrucciones repetidamente hasta que la condición evaluada deje de cumplirse, pero su implementación debe manejarse con precaución para evitar conflictos como los 'loops' infinitos.

Sintaxis

while ( condicion ){
   //codigo a ejecutar
}

let contador = 0

while ( contador <= 10 ){
  console.log(contador)
  contador ++
}



Los loops infinitos ocurren cuando la condición en un ciclo while nunca se vuelve falsa, permitiendo que el ciclo continúe indefinidamente. Esto puede llevar al consumo excesivo de recursos del sistema y, por ende, a inestabilidades en las aplicaciones. Para evitarlo: Asegúrate de modificar las variables involucradas en la condición dentro del bloque del ciclo (como por ejemplo estableciendo un contador) Esto garantiza que eventualmente el ciclo pueda salir.

Ciclo do while

(hacer mientras) crea un bucle que ejecuta una sentencia especificada, hasta que la condicion de comprobacion se evalua como falsa. la condicion se evalua despues de ejecutar la sentencia, dando como resultado que la sentencia especificada se ejecute al menos una vez.

Sintaxis

do {
   //codigo a ejecutar
}
while(condicion)

let contador = 0

do {
  console.log(contador)
  contador ++
}

while (contador <= 10)


Opta por doWhile cuando requieras que el código se ejecute al menos una vez, independientemente de si la condición es verdadera al inicio. Por otro lado, utiliza while si deseas asegurar que el código solo se ejecuta cuando la condición inicial se cumple.
