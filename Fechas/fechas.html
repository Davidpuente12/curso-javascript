Creación de Fechas

En muchas ocasiones necesitaremos guardar o trabajar con fechas en nuestros programas. Una fecha tiene datos mixtos: día, mes y año, pero también puede ser más precisa y tener hora, minutos y/o segundos. Además, la hora puede estar en varios y múltiples formatos.

JavaScript proporciona el objeto Date para manejar fechas y horas. Con él se pueden crear, manipular y formatear valores temporales.

Objeto Date

El objeto Date es nativo de JavaScript y representa un instante específico en el tiempo, medido en milisegundos desde el 1 de enero de 1970 (UTC). Su función principal es permitir la creación, manipulación y formato de fechas y horas dentro de aplicaciones web.

Lo primero, vamos a ver los constructores para saber como podemos construir una variable de tipo Date:

Constructor Descripción
new Date() Obtiene la fecha del momento actual.
new Date(str) Convierte el texto con formato YYYY/MM/DD HH:MM:SS a una fecha.
new Date(numero) Convierte el número (en formato Tiempo UNIX) a una fecha UTC.
new Date(y, m, d, h, min, s, ms) Crea una fecha UTC a partir de componentes numéricos.

// Obtenemos la fecha actual
const currentDate = new Date();

// Obtenemos la fecha 30 de Enero de 2018, a las 23h 30m 14seg
const date = new Date("2018/01/30 23:30:14");

// Obtenemos la fecha a partir de un timestamp o Tiempo UNIX
const timestamp = new Date(872817240000);

// Creamos una fecha pasando cada uno de sus componentes numéricos*
const createDate = new Date(2018, 0, 30, 23, 30, 14, 0);

En los componentes numericos, los parametros y (año) y m (mes) son obligatorios, el resto son parámetros opcionales. El parametro mes se calcula desde el 0 para enero, 1 febrero y asi sucesivamente.

Métodos para dias, meses y años

Método Descripción
getDay() Obtiene el día de la semana (0-6, domingo=0)
getFullYear() Obtiene el año (ej. 2026)
getMonth() Obtiene el mes (0-11)
getDate() Obtiene el día del mes (1-31)
getHours() Obtiene la hora, en formato militar (0-23)
getMinutes() Obtiene los minutos (0-59)
getSeconds() Obtiene los segundos (0-59)
getMilliseconds() Obtiene los milisegundos
getTime() Devuelve el UNIX Timestamp: segundos transcurridos desde 1/1/1970.
getTimezoneOffset()() Diferencia horaria (en min) de la hora local respecto a UTC.
toLocaleDateString() Devuelve la fecha en formato local
toLocaleTimeString() Devuelve la hora en formato local

Debido que Javascript no tiene forma de devolver, por ejemplo, el día de la semana o el mes en formato de texto. Podemos hacerlo de forma muy sencilla utilizando un array:

const date = new Date();

const meses = ["Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio", "Julio", "Agosto", "Septiembre", "Octubre", "Noviembre", "Diciembre",]

meses[date.getMonth()]

Formato por defecto: Fri Aug 24 2018 00:23:31 GMT+0100
Método Descripción Ejemplo de salida

Operaciones con Fechas

const inicio = new Date(2026, 2, 1);

const fin = new Date(2026, 2, 31);

const diferencia = fin - inicio;

const dias = diferencia / (1000 * 60 * 60 * 24);

Buenas Prácticas

  • Usar toLocaleDateString y toLocaleTimeString para formatos amigables.
  • Recordar que los meses empiezan en 0 (enero) y terminan en 11 (diciembre).
  • Para cálculos complejos, considerar librerías como Moment.js o date-fns.
  • Siempre validar que las fechas sean válidas para evitar errores en cálculos.
  • Evitar modificar directamente los valores internos del objeto Date; usar métodos específicos para cambios.
  • Tener en cuenta las zonas horarias y posibles cambios por horario de verano.
