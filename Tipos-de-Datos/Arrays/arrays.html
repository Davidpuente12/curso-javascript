Arrays

es una estructura de datos ordenada que permite almacenar multiples valores dentro de una variable, estos valores pueden ser cualquier tipo de dato, como otros arrays, funciones y objetos, y aunque los datos que contiene pueden ser distintos entre si, se recomienda que contengan un solo tipo de dato, debido al tipado debil de Javascript. Tambien vale la pena mencionar que los arrays son objetos.

Sintaxis

variable = [strings, numbers, boolean, objetos{}]

Corchetes [] abren un segmento en donde podemos escribir datos de cualquier tipo como un "array" (formacion), strings (cadenas), numeros, y objetos, son utiles porque permiten almacenar gran cantidad de informacion en un solo lugar.

Acceder a sus elementos

Los valores que contiene un array estan indexados por numeros empezando desde 0, esto permite acceder a sus valores mediante sus indices, Para acceder a los valores de un array, utilizamos la sintaxis variable [indice]. si el indice esta fuera de rango devuelve undefined.

Tambien es posible encontrar valores mediante el metodo at() este recibe por argumento el indice del elemento y lo devuelve, la utilidad mas importante es que recibe tambien indices negativos, algo que no se puede con las notaciones de corchetes, donde tendrias que obtener la longitud del array y restarle las unidades necesarias para encontrar elementos del final.

length es una propiedad que devuelve la longitud del arreglo, esto quiere decir que devuelve el numero de elementos que contiene. Tambien permite establecer el numero de elementos que puede tener un arreglo, esto se logra mediante la asignacion de un numero que sera la longitud maxima, haciendo que si hay elementos sobrantes se eliminen, de lo contrario se crean posiciones vacias (undefined).

Clase Array

La clase (funcion constructora) Array en JavaScript es un objeto global que sirve como plantilla para crear y manipular arreglos. Proporciona propiedades y métodos que permiten almacenar, acceder y transformar colecciones de datos de manera dinámica. Internamente, cada instancia de Array hereda métodos y propiedades de Array.prototype.

Array.isArray(array) sirve para comprobar si una variable contiene un tipo de dato array.

Efectos secundarios

El método .map() en JavaScript no modifica el array original en condiciones normales: siempre devuelve un nuevo array con los resultados de aplicar la función de callback a cada elemento. Sin embargo, hay casos concretos en los que sí puede parecer que modifica el array, y esto ocurre por efectos secundarios dentro del callback:

  • 1. Cuando el callback altera directamente el array original
    • arr.map((num, i, original) => original[i] = num * 10 );
  • 2. Cuando los elementos son objetos y se mutan sus propiedades. Aunque map crea un nuevo array, los objetos referenciados son los mismos, así que sus propiedades cambian.
    • arr.map(user => user.name = user.name.toUpperCase() );
  • 3. Cuando se usan métodos mutadores dentro del callback. Aquí el array original se ve afectado porque splice() lo modifica.
    • arr.map((num, i, original) => original.splice(i, 1); );

reduce()

"Reducir" llama a una funcion (callback) que reduce un array a un unico valor. funciona mediante un acumulador y valor inicial que acumula los elementos de un array. Util para sumar, transformar, agrupar o cualquier otra operación que necesite consolidar una colección de datos en un solo resultado.

Sintaxis

El metodo recibe dos Argumentos

La funcion que itera y reduce cada elemento del array (obligatorio), puede utilizar cuatro parametros donde los dos primeros son los mas utilizados, los otros dos son opcionales, se definen en el orden del ejemplo, si solo se define uno, estaria en la primera posicion que representa "elemento".

  • acumulador esta variable guarda el valor que se va acumulando, sino se define en "valorInicial" inicia con el valor inicial del array.
  • valorActual esta variable guarda el valor actual de array de acuerdo va iterando.
  • index esta variable guarda la posicion actual del indice, su uso es opcional.
  • array referencia al array original (opcional)

El valor inicial del acumulador que utilizará como argumento la función. Si no se especifica, en primer elemento del array se usa como acumulador inicial y la iteracion empieza en el segundo elemento.

  • reduce aqui llama al callback que recibe dos parametros principales:
    • acumulador guarda el valorInicial que establecimos en 0, a partir de este valor comenzara a acumular.
    • valorActual guara el valor actual del array segun la iteracion, es decir en la primera vuelta guara el 1, en la segunda el 2, y asi susesivamente en toda la lista.
  • arrowfunction inicia la funcion sobre estos dos primeros valores que tienen las variables antes mencionadas, sumando el acumulador con el valorActual que corresponden a 0 + 1 en la primera iteracion, el resultado = 1 ahora lo guarda la variable acumulador, itera la segunda vuelta y ahora las variables corresponden a 1 + 2 y el resultado queda en acumulador en cada vuelta hasta el final y este sera el valor que retorna.
  • El acumulador funciona de manera similar a cuando tenemos que declarar una variable que acumula por fuera de un ciclo en cada iteracion: acumulador += numero.
  • reduce recoje el arreglo "palabras" y acumula resultados en un objeto llamado "acumulador"
    • if (acumulador[valorActual]){...}
    • Aqui se verfica si la palabra "valorActual" ya existe como clave en el objeto acumulador. Si existe, se incrementa su valor(conteo). Si no existe, se inicializa con 1.

El método flatMap combina dos operaciones: aplica una función de mapeo a cada elemento de un arreglo y luego aplana el resultado en un nuevo arreglo de una sola dimensión. Es equivalente a usar map() seguido de flat(1), pero más eficiente.

Desestructuracion de Arrays

Se trata de desestructurar una estructura, que puede ser un array o un objeto. Consiste en extraer sus elementos y almacenarlos en variables individuales que deben tener nombres distintos, estos hacen referencia a las posiciones del array. Si queremos guardar posiciones especificas podemos hacerlo usando comas (,) que omiten posiciones hasta llegar al que queremos.

Spread Operator (...)

El operador de extencion es una herramienta en javascript que expande elementos de un iterable (como un array, objetos, o incluso strings) en lugares donde se esperan multiples elementos. Sirve para copiar, combinar, añadir elementos, o incluso pasar parametros a funciones, cuando se aplica en strings los expande, haciendo una separacion de cada letra. Se representa mediante tres puntos: (...).

Los array se guardan en una referencia en memoria del computador, al crear una copia esta tendra la misma referencia que el original, debido a esto, si cambia algo en la copia, tambien cambia el original. La ventaja de spread operator para copiar arrays es que guarda la copia en otra referencia de memoria.

El operador de propagacion produce una copia en un solo nivel de profundidad, esto significa que si existen objetos o arreglos dentro del array a copiar, estos sub elementos seguiran teniendo la misma refencia en memoria en la copia y el original.

Parametro rest

este parametro se usa para agrupar el residuo de elementos sobrantes que no se guardaron luego de realizar una desestructuracion o el residuo de argumentos que pasamos a una funcion, la sintaxis es (...) seguido de una variable que se suele llamar "resto", esta contendra los elementos sobrantes en un arreglo propio. El parametro simpre debera estar en la ultima posicion.

estructuredCloned()
sirve para hacer una copia profunda de un objeto o valor, incluyendo estructuras complejas como objetos anidados y arreglos, asi los cambios en el clon no afectan al original. Una copia superficial solo duplica las referencias de primer nivel. Si el objeto tiene propiedades que son otros objetos, esas referencias siguen apuntando al mismo lugar en memoria.
