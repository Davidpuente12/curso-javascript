This

this es una palabra clave especial que hace referencia al contexto de ejecución actual. Dicho de otra forma, señala quién está ejecutando el código en ese momento. Su valor depende de cómo y dónde se invoca una función, no de dónde está escrita.

  • dentro de un método de un objeto, this se refiere al objeto que contiene el método.
  • En una función global, this se refiere al objeto global (window en navegadores, global en Node.js).
  • En funciones regulares, this puede ser undefined en modo estricto o referirse al objeto global en modo no estricto.
  • En funciones flecha, this hereda el valor de this del contexto de ejecución donde se definió la función.
  • En clases: dentro de los métodos, this apunta a la instancia de la clase.

En resumen, this es una referencia dinámica que cambia según el contexto de ejecución, lo que lo hace fundamental para entender cómo funcionan los objetos y las funciones en JavaScript.

// Global
console.log(this); // En navegador: Window

// En un objeto
const persona = {
  nombre: "Luis",
  saludar() {
    console.log("Hola, soy " + this.nombre);
  }
};
persona.saludar(); // Hola, soy Luis

// Función normal
function mostrar() {
  console.log(this);
}
mostrar(); // undefined en modo estricto, objeto global en no estricto

// Función flecha
const flecha = () => {
  console.log(this);
};
flecha(); // Hereda el this del contexto donde se definió
