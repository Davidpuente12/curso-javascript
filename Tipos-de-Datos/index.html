Tipos de Datos

En Javascript tenemos 10 tipos de datos diferentes agrupados en dos categorias principales: Primitivos y complejos, estos indican el tipo de informacion que puede almacenar una variable. Son esenciales porque permiten al lenguaje saber como tratar esos datos: cómo guardarlos en memoria, qué operaciones se pueden hacer con ellos, y cómo se comportan en el programa.

typeof sirve para identificar el tipo de datos que se esta usando.

Metodos de las cadenas

  • length sirve para obtener la longitud y asi saber cuantos caracteres contiene un string, incluyendo los espacios.
  • toLowerCase() | toUpperCase() sirve para convertir strings a mayusculas o minusculas.
  • substring() sirve para extraer subcadenas, con este metodo obtenemos ciertas partes de un string, comenzando y terminando por indices especificados.
  • charAt(indice) sirve para extraer un caracter, funciona especificando la posicion numerica en el indice.
  • charCodeAt(indice) devuelve el codigo Unicode del caracter en la posicion indicada por el indice.
  • trim() elimina los espacios en blanco al principio y final de la cadena.
  • concat(variable) sirve para concatenar strings, funciona especificando el nombre de la variable que contiene el estring a concatenar.
  • match("expresion regular") sirve para buscar coincidencias, funciona especificando la expresion regular que se desea obtener de la cadena, al final devuelve el indice.
  • replace("expresion regular, reemplazo") reemplaza la primera coincidencia en el texto, funciona especificando el texto o la expresion regular y el reemplazo deseado. Para reemplazar todas las coincidencias usariamos replaceAll.
  • padStart( longitud, string ) | padEnd( longitud, string )
    este metodo aplica un "string padding", esto significa que rellena un string con un caracter especificado que podria repetirse hasta que complete la longitud maxima descrita. padStart rellena al inicio del string y padEnd lo hace al final. Recibe dos argumentos:
    • La longitud máxima a rellenar, incluyendo el string inicial.
    • El string para rellenar, por defecto, es un espacio.

Manipulacion de numeros

En JavaScript, el tipo de dato primitivo number es esencial para el manejo de operaciones numéricas. Aunque existen diversas formas de escribir números, como enteros y decimales, todos se reconocen bajo el mismo tipo de dato: number. Esta unificación facilita la manipulación y el cálculo numérico. Veamos cómo se representan en este lenguaje:

  • Operadores aritmeticos: suma (+) resta(-) multiplicacion(*) divicion(/) modulo(%) exponente(**)
  • metodo toFixed() se usa para formatear un numero con una cantidad especifica de decimales. devolviendo el resultado como una cadena de texto. El numero se redondea si es necesario.
  • Objeto math da acceso a funciones y constantes matematicas.
    • Math.PI es una propiedad del objeto global Math que representa el valor de π (pi) → 3.14159...
    • Math.round(x) es una función que redondea un número al entero más cercano. Si el decimal es 0.5 o mayor, redondea hacia arriba. Si el decimal es menor que 0.5, redondea hacia abajo. ejemplo: Math.round(4.6) → 5
    • Math.floor(x) sirve para redondear un número hacia abajo al entero más cercano. Ejemplo: Math.floor(4.6) → 4
    • Math.ceil(x) es una función que redondea un número hacia arriba al entero más cercano. Ejemplo: Math.ceil(4.2) → 5
    • Math.abs(x) devuelve el valor absoluto de x, es decir, la distancia entre ese número y el cero, sin importar el signo. - Si x es positivo o cero → lo devuelve tal cual.- Si x es negativo → lo convierte en positivo. Ejemplo: Math.abs(-7) → 7
    • Math.pow(Exponente, Potencia) sirve para elevar un número a una potencia, es decir, calcular la base elevada al exponente Potencia (x elevado a y). Alternativa moderna: (**) Ejemplo: Math.pow(2, 3) → 8
    • Math.sqrt(x) devuelve la raíz cuadrada de x, es decir, el número positivo que, al multiplicarse por sí mismo, da como resultado x. Ejemplo: Math.sqrt(9) → 3
    • Math.max(valor1, valor2...) sirve para encontrar el mayor valor entre dos o más números. Es una función muy útil cuando necesitas comparar valores y quedarte con el más grande. Ejemplo: Math.max(1, 5, 3) → 5
    • Math.min(valor1, valor2...) sirve para encontrar el menor valor entre dos o más números. Ejemplo: Math.min(1, 5, 3) → 1
    • Math.random() genera un número pseudoaleatorio entre 0 (incluido) y 1 (excluido). Es decir, cada vez que lo llamas, te da un número decimal. Ejemplo: Math.random() → 0.73...

Generar un numero aleatorio

Para generar un numero aleatorio empleamos dos metodos: Math.random() y Math.floor(). Aqui Math.random() genera un número decimal aleatorio entre 0 y 1. Multiplicamos Math.random() por 10 para obtener un rango entre 0 y 9. Luego sumamos 1 para modificar el rango entre 1 y 10. Math.floor() se utiliza para redondear hacia abajo al número entero más cercano. Podemos cambiar el rango al multiplicarlo por un numero diferente y si no sumamos el 1, el rango incluira el 0.

Conversion de Tipos

En javascript existe un concepto llamado Typecasting que hace referencia al hecho de poder convertir tipos de datos de manera explicita o implicita, La conversion implicita la realiza el lenguaje al momento de compilarse cuando sumamos tipos de datos diferentes como (2 + true = 3) el lenguaje convierte el booleano true en numero uno. La conversion explicita la realizamos manualmente con funciones del lenguaje para convertir String a number o booleanos.

  • typeof sirve para identificar el tipo de dato de una variable o expresión en tiempo de ejecución. Es muy útil para validar entradas, depurar errores o tomar decisiones condicionales según el tipo de dato.
  • ParseInt(cadena, base) es una funcion que convierte una cadena de texto en un numero entero, lo hace analizando si la cadena representa un numero. Si el argumento esta vacio devuelve NaN. La base numerica (radix) es opcional, como 10 para decimal, 2 para binario, 16 para hexadecimal.
  • ParseFloat(x) es una funcion que convierte una cadena de texto en un numero decimal, lo hace analizando si la cadena representa un numero decimal. Si el argumento esta vacio devuelve NaN.

Datos complejos

Los Tipos de datos complejos en JavaScript son más dinámicos y se utilizan para almacenar colección de valores y entidades más complejas como propiedades y metodos. Esto incluye tanto objetos como arrays y funciones.

Arrays

es una estructura de datos ordenada que permite almacenar multiples valores dentro de una variable, estos valores pueden ser cualquier tipo de dato, como otros arrays, funciones y objetos, y aunque los datos que contiene pueden ser distintos entre si, se recomienda que contengan un solo tipo de dato, debido al tipado debil de Javascript. Tambien vale la pena mencionar que los arrays son objetos.

Sintaxis

variable = [strings, numbers, boolean, objetos{}]

Corchetes [] abren un segmento en donde podemos escribir datos de cualquier tipo como un "array" (formacion), strings (cadenas), numeros, y objetos, son utiles porque permiten almacenar gran cantidad de informacion en un solo lugar.

Objetos

Los objetos en JavaScript son estructuras de datos que permiten almacenar información de manera más compleja que los arreglos o cadenas. A diferencia de estos últimos, que son sencillamente listas de elementos, los objetos están compuestos por "propiedades" y "valores" asociados.

Funciona como un almacen en donde podemos guardar "variables" y sus valores, o incluso funciones. Este concepto es crucial para representar entidades del mundo real en el ambito digital, permitiendo abstraer y estructurar informacion de forma organizada.

Sintaxis

let carro = {
  propiedad : "valor" ,
  metodo : function() {}
}

  • llaves{} abren un segmento o bloque en donde se almacenan las propiedades, valores y metodos, que representan al objeto en si.
  • Propiedad las propiedades son como una variable dentro del objeto, que pueden contener cualquier tipo de dato, estos son los valores. Cada propiedad representa un atributo del objeto.
  • Valor cualquier tipo de dato asociado a la propiedad, que puede ser un número, string, booleano, función, u otro objeto.
  • Metodos son las acciones o comportamientos que el objeto puede realizar. Estos se definen por medio de funciones.

Estructuras de datos nativas

set

Los Set en Javascript son estructuras de datos nativas que almacenan conjuntos de datos. La característica principal es que los datos insertados no se pueden repetir. por lo que si intentamos insertar un valor que ya existe, no se insertará de nuevo Para iniciar un Set, se debe crear una instancia de su clase a partir de un iterable. Generalmente, un iterable es un array.

Sintaxis

const set = new Set(iterable)

Metodos y propiedades Funcion
add(element) Añade un elemento al conjunto (si no está repetido) y devuelve el set. Muta
delete(element) Elimina el element del conjunto. Devuelve true si lo eliminó correctamente.
has(element) Comprueba si element ya existe en el conjunto. Devuelve true si existe.
clear() Vacía el conjunto completo.
size Propiedad que devuelve el número de elementos que tiene el conjunto.

const set = new Set([1,2,3,4,1])

// añadir
set.add(5)

// encadenar
set.add(2).add(6)

// eliminar
set.delete(3)

set.clear()
